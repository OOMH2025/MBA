第1章 基礎理論編：法定経営指導員講習資料の核心

企業のライフサイクル理論

企業成長の4段階と特徴

段階 特徴 組織課題 経営資源課題 活用する基礎知識
創業期 市場への参入、製品・サービスの確立 創業者中心の属人的組織 資金不足、人材不足 事業領域（ドメイン）の定義、STPマーケティング
成長期 売上・利益の急拡大 機能別組織への移行 生産能力の拡大、人材育成 組織設計の基本原則、成長戦略（アンゾフ）
成熟期 市場での地位確立、安定経営 組織の効率化、専門化 事業の多角化、新市場開拓 多角化の論理、競争地位に応じた戦略
衰退期 売上・利益の減少傾向 組織の再構築、事業転換 事業再生、新事業創出 両利きの経営、組織学習メカニズム

経営資源管理の基本概念

経営資源の4要素とその管理・活用・獲得

ヒト（人的資源）：採用、教育訓練、評価、配置転換、モチベーション管理

→ 事例I適用：組織活性化、人事制度改革の提案時に活用

モノ（物的資源）：設備、原材料、製品、立地、技術

→ 事例III適用：生産性向上、設備投資判断の根拠として活用

カネ（財務資源）：資金調達、資金配分、投資判断、財務管理

→ 事例IV適用：財務分析、投資評価の基本概念として活用

情報（情報資源）：市場情報、技術情報、顧客情報、経営情報

→ 事例II適用：マーケティング戦略、顧客分析の基盤として活用

経営資源の戦略的活用

  • 管理：既存資源の効率的運用 → 深化（Exploitation）
  • 活用：既存資源の新たな用途開発 → 両利きの経営の探索面
  • 獲得：不足資源の外部調達・内部育成 → 資源ベース戦略（RBV）

第2章 試験解法への統合アプローチ

伴走支援の考え方

従来の課題解決型から課題設定型への転換

課題解決型：専門家が課題を特定し、解決策を提示

↓

課題設定型：企業自身が課題を設定し、専門家が伴走支援

伴走支援の3つの要素

  • 対話と傾聴：企業の真の課題を引き出す → 事例での現状分析力
  • 内発的動機づけ：企業の主体性を尊重・促進 → 実行可能な提案力
  • 継続的関与：単発の助言ではなく継続的な支援 → PDCAサイクルでの改善提案

情報通信技術による市場環境変化

「規模の経済」から「価値の経済」への転換

規模の経済時代：大量生産・大量販売による競争優位

↓

価値の経済時代：個別ニーズへの対応、付加価値創造による競争優位

中小企業への影響

  • 市場アクセスの向上：ECプラットフォームによる全国・海外市場への参入 → 事例II：オムニチャネル戦略、デジタルマーケティング提案
  • 顧客との直接関係構築：SNS、クラウドファンディング等による顧客との直接コミュニケーション → 事例II：CRM構築、ブランド戦略立案
  • 製造・販売の効率化：クラウドサービス、IoT、AIによる業務効率化 → 事例III：生産管理システム、品質管理改善提案

百者百様の経営と統計的傾向

企業の多様性と個別性の尊重

統計的な傾向やベストプラクティスを理解しつつも、各企業固有の状況（業種、規模、地域、経営者の価値観等）を十分に考慮した支援が必要。画一的なソリューションの押し付けではなく、企業の特性に応じたカスタマイズが重要。

→ 事例解答への応用：設問で与えられた企業の特徴（業種、規模、地域性、経営方針等）を必ず考慮し、その企業に最適化された提案を行う。

事例III：生産戦略の総合的判断

レイヤー区分での体系的分析

レイヤー 分析視点 活用理論 適用する基礎知識
経営戦略レイヤー 投資効果測定による戦略妥当性 経営資源の獲得・活用理論 正味現在価値（NPV）法、内部収益率（IRR）法
生産管理レイヤー 制約条件下での最適生産計画 法的制約の理解と活用 生産管理の基本目標（PQCDSME）、受注生産と見込生産
生産性改善レイヤー 持続的改善の仕組み構築 継続的支援の考え方 改善の原則（ECRS）、IE（方法研究と作業測定）

投資判断の論理構造

補助金は投資負担軽減であり、投資内容が不適切であれば経営悪化を招く

  1. 投資効果の独立検証：補助金なしでも投資価値があるか → 基礎知識：貨幣の時間的価値、正味現在価値（NPV）法
  2. 補助金による実現可能性向上：資金調達リスクの軽減 → 基礎知識：回収期間法による投資判断
  3. リスク軽減策の検討：投資回収期間の短縮、代替案の準備 → 基礎知識：損益分岐点分析（CVP分析）

事例IV：財務分析の実務的応用

財務指標の統合的解釈

収益性指標

総資本経常利益率、売上高営業利益率、損益分岐点比率

→ 基礎知識活用：損益分岐点分析で売上高と利益の関係を把握し、収益改善の具体策を提示

安全性指標

自己資本比率、流動比率、固定長期適合率

→ 基礎知識活用：キャッシュフローの区分で資金繰りの健全性を評価

成長性指標

売上高成長率、利益成長率、設備投資比率

→ 基礎知識活用：企業のライフサイクル理論で成長段階に応じた投資戦略を評価

効率性指標

総資本回転率、在庫回転率、売掛金回転率

→ 基礎知識活用：経営分析の視点で資産効率の改善余地を特定

資金調達の最適化理論

内部資金

利益留保、減価償却費

  • メリット：金利負担なし、返済義務なし
  • デメリット：調達額に限界、成長機会の逸失

外部資金

借入、出資

  • 借入：金利負担あり、返済義務あり、レバレッジ効果
  • 出資：配当負担、経営参加権、資本コスト

第4章 解答作成プロセスでの活用

解答メモ作成段階での理論活用

与件文から抽出した要素を講習資料の理論体系で整理

  • 現状分析：企業ライフサイクルのどの段階か → 基礎知識：組織の発展段階と課題で典型的な課題パターンを想起
  • 課題特定：経営資源の「管理・活用・獲得」のどこに問題があるか → 基礎知識：内外環境分析（SWOT）で課題の構造化
  • 対策立案：伴走支援の観点から実行可能な提案は何か → 基礎知識：両利きの経営で既存改善と新規探索のバランス検討

解答作成段階での「お作法」適用

結論先行の構造化

基本構造

結論（提案）→理由（根拠）→具体策（実行方法）

例

「内発的動機づけによる人事制度改革を提案する。理由は従業員の主体性向上が組織活性化に不可欠だからである。具体的には評価制度の見直しと教育研修の充実を図る。」

制約文字数での効率的表現

キーワード活用例

  • 「ライフサイクル転換期における○○戦略」
  • 「経営資源の多重利用による××効果」
  • 「内発的動機づけによる組織活性化」
  • 「伴走支援による継続的改善」

具体性確保の方法

「百者百様」の視点を活用した具体化

  • 業種特性：製造業なら品質管理、サービス業なら顧客満足度
  • 規模特性：小規模なら経営者の個性、中規模なら組織運営
  • 地域特性：都市部なら競争環境、地方なら地域密着
  • 成長段階：創業期なら市場確立、成熟期なら差別化

第5章 PDCAサイクルでの継続的改善

Plan：課題設定での活用

課題設定型アプローチの実践

  • 画一的な解法パターンの適用ではなく、企業固有の課題設定
  • 理論の機械的適用ではなく、企業特性に応じたカスタマイズ
  • 短期的効果だけでなく、長期的持続性を考慮した目標設定

基礎知識の活用：

  • 事業領域（ドメイン）の定義：企業の存在意義から課題を設定
  • 競争地位の源泉：外部ポジショニングか内部資源かで課題の焦点を決定

Do：演習での検証ポイント

  1. 理論適用の適切性：企業のライフサイクル段階と提案内容の整合性
  2. 実現可能性：経営資源制約と法的制約の両面からの検証
  3. 政策整合性：個別企業利益と社会的意義の両立
  4. 継続性：単発の施策ではなく、継続的改善の仕組み

Check：振り返りでの改善視点

  • 画一化チェック：「百者百様」の視点を忘れていないか
  • 短期志向チェック：長期的持続性を考慮しているか
  • 法的リスクチェック：関連法令への適合性は確保されているか
  • 実行可能性チェック：経営資源の制約を適切に考慮しているか

Action：対応策の実行

継続的改善のサイクル構築

  • 理論理解の深化：講習資料の再読と理解の精緻化
  • 適用パターンの蓄積：企業タイプ別の理論適用事例の整理
  • 表現技術の向上：キーワード活用と文章構成の改善

基礎知識の継続的活用：

  • 組織学習メカニズム：シングルループ学習（手法改善）とダブルループ学習（前提見直し）
  • 両利きの経営：既存手法の深化と新手法の探索

第6章 試験当日の実践的活用

時間配分と優先順位

80分の戦略的配分

  • 読解・分析（20分）：ライフサイクル段階の特定、経営資源の棚卸し
  • 構想・企画（30分）：伴走支援の視点で実行可能な提案の構築
  • 記述・推敲（25分）：理論キーワードを活用した簡潔な表現
  • 見直し（5分）：法的制約と実現可能性の最終確認

設問別アプローチパターン

事例I：組織・人事

  • Step1：ライフサイクル段階の特定 → 組織の発展段階と課題
  • Step2：組織課題の構造化 → 組織設計の基本原則
  • Step3：人材活用策の提案 → モチベーション理論、人的資源管理の一貫性

事例II：マーケティング

  • Step1：市場環境の変化認識 → 規模の経済→価値の経済
  • Step2：顧客・競合分析 → STPマーケティング、競争地位に応じた戦略
  • Step3：マーケティングミックス → 4P統合運用、IMC

事例III：生産・技術

  • Step1：生産課題の体系化 → 生産管理の基本目標（PQCDSME）
  • Step2：改善手法の選択 → 改善の原則（ECRS）、IE手法
  • Step3：投資効果の検証 → NPV法、損益分岐点分析

事例IV：財務・会計

  • Step1：財務状況の診断 → 経営分析の視点（収益性・安全性・効率性）
  • Step2：資金調達の最適化 → 内部資金vs外部資金
  • Step3：投資判断の論理 → 貨幣の時間的価値、IRR法

危機管理とリカバリー

想定外事態への対応

  • 時間不足の場合：理論キーワードを軸とした骨子のみでも記述
  • 解釈困難な場合：「百者百様」を前提とした複数解釈の併記
  • 知識不足の場合：伴走支援の基本姿勢（対話・傾聴・継続）で補完

第7章 重要法令と制約条件

労働関連法令

労働基準法の要点

  • 労働時間：週40時間、1日8時間の原則 → 事例I適用：働き方改革、生産性向上提案時の制約条件
  • 休日：週1日以上の休日確保 → 事例III適用：生産計画立案時の制約条件
  • 割増賃金：時間外25%以上、休日35%以上、深夜25%以上 → 事例IV適用：コスト計算時の考慮要素

同一労働同一賃金

正規・非正規間の不合理な待遇格差の禁止

→ 事例I適用：人事制度改革提案時の必須考慮事項

会社法・商法関連

株主総会・取締役会

  • 重要事項の決定：定款変更、合併、重要な財産の処分等 → 事例I適用：組織再編提案時の手続き要件
  • 利益相反取引：取締役会での承認が必要 → 事例IV適用：関連会社取引の適法性確保

独占禁止法関連

競争制限行為の禁止

  • 価格協定：競合他社との価格取り決めの禁止 → 事例II適用：価格戦略提案時の制約条件
  • 市場分割：競合他社との市場・顧客の分割禁止 → 事例II適用：市場戦略立案時の注意事項
  • 不公正な取引方法：優越的地位の濫用、排他的取引等の禁止 → 事例II・III適用：取引条件設定時の制約

環境・安全関連法令

環境保護法令

  • 大気汚染防止法：排出基準の遵守 → 事例III適用：生産設備投資時の環境配慮
  • 水質汚濁防止法：排水基準の遵守 → 事例III適用：製造プロセス改善時の制約条件
  • 廃棄物処理法：適正な廃棄物処理 → 事例III適用：循環型生産システム構築時の要件

労働安全衛生法

  • 安全衛生管理体制：一定規模以上の事業場での管理者選任義務 → 事例I・III適用：組織体制・生産体制構築時の必須要件
  • 機械等の安全基準：設備の安全確保義務 → 事例III適用：設備投資・レイアウト変更時の制約

設問解釈段階

✓ ライフサイクル段階の特定

✓ 経営資源の棚卸し

  • □ ヒト：技能、経験、モチベーション状況は
  • □ モノ：設備、技術、立地の活用状況は
  • □ カネ：資金力、投資余力の状況は
  • □ 情報：市場情報、ノウハウの蓄積状況は

✓ 制約条件の確認

  • □ 労働法制：労働時間、安全衛生の制約は
  • □ 会社法制：意思決定プロセスの要件は
  • □ 競争法制：独禁法、下請法の制約は
  • □ 環境法制：環境基準、廃棄物処理の制約は

解答構築段階

✓ 伴走支援の視点

  • □ 対話・傾聴：企業の真の課題を把握したか
  • □ 内発的動機：企業の主体性を尊重した提案か
  • □ 継続性：単発ではなく持続的な改善提案か

✓ 百者百様の配慮

  • □ 業種特性：製造業・サービス業等の違いを考慮
  • □ 規模特性：従業員数、売上規模に応じた提案
  • □ 地域特性：都市部・地方の立地条件を考慮
  • □ 経営方針：経営者の価値観、経営理念を尊重

✓ 政策目的との整合

  • □ 短期利益：企業の直接的効果（売上・利益向上）
  • □ 長期持続：企業の持続的成長への貢献
  • □ 社会貢献：地域経済、雇用創出への効果
  • □ 産業発展：業界全体の競争力向上への寄与

解答記述段階

✓ 理論キーワードの効果的活用

  • □ ライフサイクル：「創業期における○○」「成熟期での××」
  • □ 経営資源：「ヒトの活用による△△」「情報資源の獲得で□□」

  • □ 基礎理論：「STPマーケティング」「SWOT分析」「RBV」等

✓ 文章構成の最適化

  • □ 結論先行：提案→理由→具体策の順序
  • □ 簡潔性：制限文字数内での要点整理
  • □ 具体性：抽象論ではなく実行可能な内容
  • □ 一貫性：設問間での論理的整合性

最終確認段階

✓ リスク回避チェック

  • □ 法的適合性：関連法令への抵触はないか
  • □ 実現可能性：経営資源制約は考慮されているか
  • □ 倫理的適切性：社会的責任は果たされているか
  • □ 持続可能性：長期的な実行可能性はあるか

✓ 加点要素の確認

  • □ 独創性：他の受験生との差別化要素
  • □ 統合性：複数の理論を統合した提案
  • □ 実務性：実際の現場で使える内容
  • □ 発展性：将来の展開可能性を示唆

第10章 基本用語・キーワード辞典

企業経営理論

戦略・ドメイン関連

  • 経営資源: 企業の活動基盤となるヒト・モノ・カネ・情報などの要素
  • ドメイン: 企業が事業を行う活動領域。「誰に、何を、どのように」提供するかを定義
  • SWOT分析: 内部環境の強み(S)・弱み(W)と、外部環境の機会(O)・脅威(T)を分析する手法
  • 3C分析: 顧客(Customer)・競合(Competitor)・自社(Company)を分析する手法
  • PEST分析: マクロ環境の政治(P)・経済(E)・社会(S)・技術(T)を分析する手法
  • リソースベースドビュー (RBV): 企業の内部資源が競争優位の源泉と考える理論
  • VRIO分析: 経営資源の価値(V)・希少性(R)・模倣困難性(I)・組織(O)を評価するフレームワーク
  • コアコンピタンス: 競合他社に真似されにくい、企業の中核的な強み
  • ケイパビリティ: 企業の組織的な能力
  • アウトソーシング: 自社の業務の一部を外部に委託すること

成長戦略関連

  • アンゾフの成長ベクトル: 市場浸透、新製品開発、新市場開拓、多角化の4つの成長戦略の方向性を示すフレームワーク
  • 多角化: 企業が新たな製品や市場に進出すること。関連多角化と無関連多角化がある
  • シナジー（相乗効果）: 複数の要素が組み合わさることで、単独よりも大きな効果を生むこと
  • 範囲の経済: 複数の事業で経営資源を共有することでコスト削減や効率化を図る効果

組織設計関連

  • 組織構造の設計原理: 専門化の原則、権限責任一致の原則、統制範囲の原則、命令統一性の原則、例外の原則（権限委譲の原則）など
  • 組織のライフサイクル: 組織が経験する起業者段階、共同体段階、公式化段階、精巧化段階の発展段階
  • 官僚制の逆機能: 形式主義、目的の置換、繁文縟礼、セクショナリズムなど、官僚制組織が生み出す非効率な側面
  • コンフリクト: 組織内の対立や利害の不一致。解消方法として競争、回避、妥協、協力（協調）、適応がある
  • 機能別組織: 製造・営業・開発といった職能ごとに部門を編成する組織形態
  • 事業部制組織: 製品別・地域別などの事業単位で分権化された組織形態
  • マトリクス組織: 職能と事業（プロジェクト）など、二元的な指揮命令系統を持つ組織形態
  • タスクフォース／プロジェクトチーム: 特定課題解決のために各部門からメンバーを集めた一時的または長期的な組織

組織文化・学習関連

  • 組織文化: 組織メンバーで共有される価値観、信念、行動パターン。シャインの3水準モデル（人工物、価値観、基本的仮定）が有名
  • 組織学習: 組織が経験から学び行動を改善するプロセス。シングルループ学習とダブルループ学習がある
  • 両利きの経営: 既存事業の深化（知の深化）と新規事業の探索（知の探索）を同時に行う経営

人的資源管理関連

  • モチベーション理論: 人の意欲を引き出す要因（内容理論）やプロセス（過程理論）を説明する理論群
    • 欲求段階説（マズロー）: 人間の欲求を5段階の階層で捉える
    • ERG理論（アルダファー）: 欲求を生存・関係・成長の3つに分類
    • 二要因理論（ハーズバーグ）: 満足に関わる動機付け要因と不満足に関わる衛生要因を区別
    • 期待理論: 努力が成果に、成果が報酬に繋がるという期待と報酬の魅力で動機づけられるとする
  • 目標管理制度 (MBO): 個人の目標と組織目標を連鎖させ、自主的な達成を促す管理手法
  • コンピテンシー: 高い成果を上げる人材の行動特性
  • OJT (On-the-Job Training): 職場での実務を通じた教育訓練
  • Off-JT (Off-the-Job Training): 職場を離れて行われる教育訓練

マーケティング関連

  • ターゲットマーケティング: 市場を細分化(S)し、標的市場を選定(T)し、製品を位置づける(P)マーケティング手法
    • セグメンテーション: 市場を共通ニーズで小グループに分割すること
    • ターゲティング: 狙うべき市場セグメントを選び出すこと
    • ポジショニング: 競合と比べ自社製品を顧客の心の中で独自に位置づけること
  • マーケティングミックス (4P): 製品(P)・価格(P)・流通(P)・プロモーション(P)の組み合わせ
  • ブランド: 自社製品・サービスを他社と識別・差別化する名称やデザインなど
  • ブランドエクイティ: ブランドが持つ資産価値（認知、ロイヤルティ、品質知覚、連想など）
  • ブランド拡張: 既存ブランド名を新カテゴリー製品に利用する戦略
  • ライン拡張: 既存ブランド名を同カテゴリー内新製品に利用する戦略
  • マルチブランド戦略: 同一カテゴリーに複数ブランドを投入する戦略

価格戦略関連

  • スキミングプライシング（初期高価格戦略）: 新製品導入時に高価格を設定する戦略
  • ペネトレーションプライシング（初期低価格戦略）: 新製品導入時に低価格を設定する戦略
  • EDLP (Everyday Low Price): 特売をせず年間を通じ低価格で販売する政策

流通・販促関連

  • チャネル戦略: 製品を消費者に届ける経路の設計・管理戦略。開放的、選択的、排他的チャネルがある
  • VMS（垂直的マーケティングシステム）: 生産から販売までの流通チャネルが統合・調整されたシステム
  • オムニチャネル: 複数チャネルを連携させ一貫した顧客体験を提供する戦略
  • プロモーションミックス: 広告、人的販売、販売促進、パブリシティなどの組み合わせ
  • プッシュ戦略: 中間業者への働きかけで製品を消費者に押し出す戦略
  • プル戦略: 消費者への訴求で製品を求めてもらう戦略
  • IMC（統合型マーケティングコミュニケーション）: 様々な顧客接点で一貫したメッセージを発信する手法
  • AIDMA / AISAS: 消費者の購買行動プロセスモデル

財務会計

キャッシュフロー関連

  • 営業キャッシュフロー (営業CF): 企業の本業から生み出される現金の増減額
  • 投資キャッシュフロー (投資CF): 固定資産や有価証券の取得・売却など、将来の利益獲得のための投資活動による現金の増減額
  • 財務キャッシュフロー (財務CF): 借入れや返済、株式発行など、資金調達・返済活動による現金の増減額
  • フリーキャッシュフロー (FCF): 企業が事業活動から生み出し、債権者と株主が自由に使える現金
  • 運転資本: 売上債権と棚卸資産の合計から仕入債務を差し引いたもの。事業運営に必要な短期的な資金

収益性指標

  • 売上高総利益率 (粗利益率): 売上高に対する売上総利益の割合
  • 売上高営業利益率: 売上高に対する営業利益の割合
  • 売上高経常利益率: 売上高に対する経常利益の割合

効率性指標

  • 有形固定資産回転率: 売上高を有形固定資産で割った値
  • 棚卸資産回転率: 売上高（または売上原価）を棚卸資産で割った値
  • 売上債権回転率: 売上高を売上債権で割った値

安全性指標

  • 自己資本比率: 総資産に占める自己資本の割合
  • 負債比率: 自己資本に対する負債の割合
  • 固定比率: 自己資本に対する固定資産の割合
  • 固定長期適合率: 自己資本と固定負債の合計に対する固定資産の割合
  • 流動比率: 流動負債に対する流動資産の割合
  • 当座比率: 流動負債に対する当座資産の割合

損益分岐点分析

  • 損益分岐点売上高: 費用と収益が等しくなり、利益がゼロとなる売上高
  • 限界利益: 売上高から変動費を差し引いた利益
  • 変動費: 売上高の増減に比例して変動する費用
  • 固定費: 売上高の増減に関わらず一定期間発生する費用
  • 安全余裕率: 実際の売上高が損益分岐点売上高をどれだけ上回っているかを示す割合
  • 営業レバレッジ: 固定費の存在により、売上高の変化が営業利益に何倍の影響を与えるかを示す度合い

投資評価

  • 割引率 (r): 将来のキャッシュフローを現在の価値に割り引く際に用いる率
  • 現在価値 (PV): 将来受け取るキャッシュフローを、割引率を用いて現在の価値に換算したもの
  • 将来価値 (FV): 現在の金額が、一定の利率で複利運用された場合の将来の価値
  • 正味現在価値 (NPV): 投資によって将来得られるキャッシュフローの割引現在価値の合計から、初期投資額を差し引いたもの
  • 内部収益率 (IRR): 投資のNPVがゼロになる割引率
  • 収益性指数 (PI): 各年のキャッシュフローの現在価値の合計を初期投資額で割った値
  • 回収期間法 (PP法): 投資額を何年で回収できるかを計算する方法
  • 割引回収期間法 (DPP法): キャッシュフローを現在価値に割り引いた上で回収期間を計算する方法

運営管理

生産管理全般・改善

  • リードタイム: 発注から納品まで、あるいは生産着手から完成までの所要時間
  • 生産リードタイム: 生産に着手してから製品が完成するまでの時間
  • 3S (生産の合理化): 単純化(Simplification)・標準化(Standardization)・専門化(Specialization)の総称
  • 5S: 職場の管理改善の基本。整理・整頓・清掃・清潔・躾
  • ECRSの原則: 改善策を検討する際の視点。排除(Eliminate)・結合(Combine)・交換(Rearrange)・簡素化(Simplify)
  • 付加価値: 自社の活動の結果として新たに付与された価値
  • 歩留まり: 投入された原材料量に対して、産出された品物の量の割合
  • 稼働率: 人や設備が実際に稼働していた時間の割合
  • QCサークル: 品質管理や職場改善のための自主的な小グループ活動
  • ZD (Zero Defects) 運動: 製品不良ゼロを目指す活動
  • 多能工: 複数の異なる種類の作業や工程をこなせる作業者
  • IE (インダストリアル・エンジニアリング): 生産性向上を目的とし、方法研究や作業測定を行う科学的管理手法

生産方式・計画・統制

  • 個別生産: 個々の注文に応じてその都度生産する形態
  • ロット生産: 品種ごとに生産量をまとめて一定期間続けて生産する形態
  • 連続生産: 特定の製品を大量に連続して生産する形態
  • 受注生産: 顧客からの注文を受けてから製品を生産する方式
  • 見込生産: 需要を予測して事前に製品を生産する方式
  • ロットサイズ: 1回に生産または発注する単位量
  • 段取り替え: ある製品の生産終了後、次の異なる製品の生産準備をする作業
  • 内段取り／外段取り: 内段取りはラインを停止して行う段取り替え、外段取りはラインを停止せずに行える段取り替え

レイアウト・生産システム

  • 機能別レイアウト (工程別レイアウト): 同じ種類の機械や設備を一箇所に集めて配置する工場レイアウト
  • 製品別レイアウト: 原材料から製品完成までの工程順に機械や設備を配置する工場レイアウト
  • ライン生産方式: 生産ライン上で製品が移動しながら加工が進む生産方式
  • セル生産方式: 複数の異なる機械設備をU字型などにまとめ、作業者が多工程を担当する生産方式
  • ラインバランシング: ライン生産方式で各作業ステーションの作業時間を均等化すること
  • サイクルタイム (ピッチタイム): ライン生産方式で、1つの製品を生産するのに要する時間、または製品がラインから1つずつ出てくる時間間隔

トヨタ生産方式

  • トヨタ生産方式 (JIT, 自働化): ジャストインタイムと自働化を2本柱とする生産システム
  • JIT (ジャストインタイム): 必要なものを、必要なときに、必要なだけ生産・運搬する考え方
  • かんばん: JITを実現するための道具で、後工程が前工程に必要な部品の種類や量を指示する札
  • 自働化 (ニンベンのついたジドウカ): 機械が異常を検知すると自動的に停止し、不良品を作らないようにする考え方
  • あんどん: 生産ラインの異常を知らせる表示灯

生産計画・統制

  • 生産計画: 何を、いつまでに、どれだけ生産するかを計画すること。大日程・中日程・小日程計画がある
  • 生産統制: 生産計画通りに生産が進むように管理・調整する活動
    • 作業手配 (差立): 次にどの作業を行うかを決定し指示すること
    • 余力管理: 生産能力と生産負荷の差（余力または不足）を把握し調整すること
    • 進捗管理: 生産計画と実績の差異を把握し、遅れがあれば対策を講じること
    • 現品管理: 材料・部品・仕掛品・完成品の数量や保管場所を正確に把握すること
  • MRP (資材所要量計画): 生産計画に基づき、必要な部品や材料の量を計算し、発注時期や量を決定するシステム
  • BOM (部品表): 製品を構成する部品や原材料の一覧とその必要量を示した表

IE・時間研究

  • 工程分析: 製品の製造工程や作業の流れを図記号で表し、ムダを発見・改善する手法
  • 動作研究: 作業者の動作を分析し、疲労を軽減し効率的な作業方法を追求する手法
  • サーブリッグ分析: 作業動作を18の基本要素（動素）に分解して分析する手法
  • 稼働分析: 作業者や機械の稼働状況を調査し、非稼働時間（手待ちなど）の原因を分析する手法
  • 時間研究: 作業要素ごとの時間を測定し、標準時間を設定する手法
  • 標準時間: 標準的な熟練度の作業者が、所定の作業方法と設備で、標準的なペースで作業を行うのに必要な時間

生産情報システム

  • CAD (Computer-Aided Design): コンピュータ支援設計システム
  • CAM (Computer-Aided Manufacturing): コンピュータ支援製造システム
  • CAE (Computer-Aided Engineering): コンピュータ支援エンジニアリングシステム
  • CNC (Computerized Numerical Control): コンピュータ数値制御
  • FMC (Flexible Manufacturing Cell): 複数のNC工作機械やロボットなどを組み合わせ、多様な部品を柔軟に加工できる生産単位
  • FMS (Flexible Manufacturing System): FMCを複数組み合わせ、自動搬送装置や集中管理システムで統合した、より大規模で柔軟な生産システム
  • CIM (Computer Integrated Manufacturing): 設計・生産準備・生産・販売・在庫管理など、企業の生産活動全体をコンピュータで統合管理するシステム
  • コンカレントエンジニアリング (CE): 製品開発の各工程（設計、生産準備など）を同時並行で進め、開発期間短縮やコスト削減を図る手法

店舗・販売管理

  • ISM (インストア・マーチャンダイジング): 店舗内での販売促進活動の総称
  • スペースマネジメント: POSデータなどに基づき、棚割り（プラノグラム）を最適化し、売上や利益を最大化する売場管理手法
  • プラノグラム: 具体的な商品の棚割り計画図
  • クロスマーチャンダイジング (CMD): 関連する異なるカテゴリーの商品を組み合わせて陳列・提案し、合わせ買いを促す手法
  • マーケット・バスケット分析: 顧客が同時に何を購入したか（バスケットの中身）を分析し、併売されやすい商品の組み合わせを見つけ出す手法

第12章 実践的活用のまとめ

本ガイドの特徴と活用効果

理論と実践の統合

法定経営指導員講習資料の理論的基盤と、二次試験の実践的解法技術を統合。単なる暗記ではなく、企業の実情に応じた柔軟な理論適用力を養成する構成となっています。

伴走支援思考の習得

従来の「専門家が解決策を提示」から「企業と共に課題を設定し解決を図る」へのパラダイム転換。この思考法により、実務で通用する診断士としての基盤が形成されます。

法的制約の体系的理解

提案の実現可能性を左右する重要法令を体系化。適法性を担保した実行可能な提案力により、安定した得点確保が可能になります。

継続的スキル向上への道筋

段階的習得プロセス

  1. 理論理解段階：基礎知識の体系的理解と講習資料の深読
  2. 適用練習段階：過去問演習での理論適用と解法パターン習得
  3. 統合発展段階：複数理論の統合的活用と独自解法の確立
  4. 実践応用段階：実務を見据えた診断・助言能力の完成

本ガイド活用後の到達目標

  • 理論的裏付けのある提案力：根拠ある診断・助言の実施
  • 企業特性に応じた柔軟性：画一的でない個別最適化対応
  • 法的制約を考慮した実現可能性：実務で通用する提案品質
  • 継続的改善への導き：単発でない持続的支援の実現

試験合格後の展望

実務での活用可能性

本ガイドで習得した理論体系と伴走支援の考え方は、試験合格後の実務診断においても直接活用可能。特に、中小企業の個別性を尊重しつつ体系的な支援を行う能力は、実務家診断士として不可欠なスキルです。

社会的貢献への道筋

個別企業支援を通じた地域経済活性化、雇用創出、産業競争力向上など、診断士の社会的使命を果たすための基盤が本ガイドによって確立されます。

最終メッセージ

中小企業診断士二次試験は、単なる知識の暗記試験ではありません。企業の実情を理解し、適切な診断と実行可能な助言を行う能力を問う試験です。本ガイドを活用し、理論に裏打ちされた実践的診断能力を身につけ、試験合格はもちろん、その先の実務家診断士としての活躍を目指してください。

頑張る全ての受験生の合格と、その後の活躍を心より祈念いたします。

本ガイドは法定経営指導員講習資料を基に、中小企業診断士二次試験対策として編纂されました。
